What you need to learn to become a web developer

To become a web developer you have to learn some technologies. If you have no experience you can start with the following list:

  1. HTML

    2. Learn HTML to structure a simplest web page. It is the base of the remaining.

  2. CSS

    3. CSS is used to stylize the page.

  3. JavaScript

    4. JavaScript is used to provide a great user experience.

  4. PHP

    5. PHP is a server side scripting language that is embedded in HTML

  5. MySQL

    6. MySQL is a relational database management system.
